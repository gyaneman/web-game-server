# web-game-server
現在制作中!!

Webでオンラインゲームを作ります。
どんなゲームを作るかはまだ決定していませんが、オセロのようなゲームになると思います。

## 環境構築～実行
あらかじめ以下のことを
* [Node.js](https://nodejs.org/en/)インストール
* [mongoDB](https://www.mongodb.org/)をインストール
* mongoDB起動

あとは以下のコマンドを実行してください。
```
$ git clone https://github.com/gyaneman/web-game-server.git
$ npm install
$ gulp
$ mongo
  use web-game-server
$ npm start
```
